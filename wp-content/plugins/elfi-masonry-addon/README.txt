"Elfi Masonry" is a  filterable & Gallery masonry addon for elementor. This plugin provides powerful Gallery masonry & beautiful animated filtering with Category Based.

Elfi has amazing features, attractive layout design, amazing buttons style, popup image, and more than 40+ control settings that allow the user to customize easily.



**Free Version Features**

* Gallery Masonry
* Filter Masnry (Category based filter)
* Portfolio Filter
* Post Filter
* Allows to change the permalink 'elfi' [view details](https://elfi.sharabindu.com/wp/docs/chanage-the-elfi-permalink/)
* Fully responsive design
* Popup Image
* 2,3, and 4 column layouts
* Read more button
* Super customized addon
* All browsers supported
* Clean coding, validate W3C
* Well documented

== Installation ==

Go to the Plugins Menu in WordPress and select Plugins > Add new
Search for “Elfi Masonry Filter”
Click “Install” and “activate”.
